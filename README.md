# 🚀 Rejouice – Creative Agency Website Clone

A modern, animated static website inspired by Rejouice, built using **HTML, CSS, and JavaScript**, with smooth scrolling and advanced animations powered by **GSAP**, **ScrollTrigger**, and **Locomotive Scroll**.

---

## ✨ Features

* 🎯 Smooth scrolling using Locomotive Scroll
* 🎬 Scroll-based animations with GSAP & ScrollTrigger
* 🎨 Modern UI inspired by creative agencies
* 📱 Fully responsive layout
* 🎥 Video integration for interactive sections
* 🖱 Custom cursor interactions

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* GSAP (GreenSock Animation Platform)
* ScrollTrigger
* Locomotive Scroll
* Remix Icons

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/nafy-max/Rejoiuce.git
```

2. Navigate into the project folder:

```bash
cd Rejoiuce
```

3. Open `index.html` in your browser
   (or use Live Server in VS Code for best experience)
